# Nuevo Proyecto de Investigación
# Usa esta carpeta como plantilla para agregar más proyectos

## Pasos:
1. Copia esta carpeta y renómbrala con el slug de tu proyecto (ej: `ml-redes-2026`)
2. Edita el `metadata.json` con la información del proyecto
3. Sube tus archivos PDF
4. Ve a Supabase → Table Editor → research_projects → Insert row con el mismo slug

## Template metadata.json:
```json
{
  "title": "Título de tu investigación",
  "slug": "slug-unico-del-proyecto",
  "area": "IA / Ciberseguridad / TIS / Redes",
  "status": "planning",
  "progress": 0,
  "abstract": "Resumen de la investigación...",
  "keywords": ["keyword1", "keyword2"],
  "institution": "UNEMI",
  "year": 2026
}
```
